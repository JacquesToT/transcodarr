# Contributing to Transcodarr

Bedankt voor je interesse in het bijdragen aan Transcodarr!

## Development Setup

1. Fork de repository
2. Clone je fork: `git clone https://github.com/jouw-username/transcodarr.git`
3. Maak een feature branch: `git checkout -b feature/jouw-feature`
4. Test je wijzigingen grondig
5. Commit met duidelijke boodschappen
6. Push naar je fork
7. Open een Pull Request

## Code Style

- **Shell Scripts**: Volg Google Shell Style Guide
- **Python Code**: PEP 8 compliance (monitor applicatie)
- Gebruik descriptieve variabele namen
- Voeg comments toe voor complexe logica

## Testing

Test alle wijzigingen op:
- Synology NAS (Docker container)
- Apple Silicon Mac (transcode node)

## Bug Reports

Open een GitHub issue met:
- Duidelijke beschrijving van het probleem
- Stappen om te reproduceren
- Verwacht vs. actueel gedrag
- System informatie (OS, versies)
